# nanoc-bluecloth

This provides a [BlueCloth](http://deveiate.org/projects/BlueCloth) filter for [nanoc](http://nanoc.ws).

## Installation

`gem install nanoc-bluecloth`

## Usage

```ruby
filter :bluecloth
```

This filter takes no options.
